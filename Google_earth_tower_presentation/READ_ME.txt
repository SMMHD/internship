To run the create_kmz.py script, you need to install the following libraries:





Pillow: For processing images and extracting EXIF data



simplekml: For creating KMZ files

Installation instructions:





Open a terminal or command prompt.



Ensure Python is installed (check by running: python --version).



Run the following command to install both libraries: pip install Pillow simplekml

Note: If you're using a virtual environment, activate it first: venv\Scripts\activate Then run the installation command above.

After installation, you can run the Google_earth_tower_presentation.py script.
